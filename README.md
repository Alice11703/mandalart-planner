# 만다라트 플래너 (Mandarat-Plans)

## 소개
만다라트 플래너는 목표 설정과 달성을 돕는 온라인 계획 도구입니다. 핵심 목표를 중심으로 세부 목표를 체계적으로 수립하고 관리할 수 있습니다.

## 주요 기능
- 8x8 만다라트 보드 생성 및 관리
- 목표 진행 상황 트래킹
- 데이터 자동 저장
- 반응형 디자인

## 기술 스택
- Frontend: Next.js, TypeScript
- Styling: Tailwind CSS, Emotion
- Deployment: Vercel
